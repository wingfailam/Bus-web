# Bus

This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 12.2.12.

## 功能
* 任選縣市、公車路線、往返方向以取得所選站點資訊及預估到站時間
* 地圖呈現公車路線、其站序、與行駛中的公車位置
* 點擊地圖上的站點將顯示該站點各路線之預估到站時間（依時間排序）
* 點擊地圖上的公車將顯示車牌號碼
* 以三十秒為期自動更新，亦可手動更新
* 支援RWD
